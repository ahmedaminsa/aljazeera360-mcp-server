# Helpful SQL for Key Metrics

Using the Data Sync feeds above ingested into your data warehouse or databased you can run the SQLs below for key metrics that might be helpful to your business, as well as demonstrate how to use the feeds.

## Subscription/PPV Metrics

| Metric                                              | SQL                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| --------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Total New Successful Purchases, by licence type     | <p><strong>select</strong> </p><p>licence\_type , count(order\_id) as total\_new\_succ\_purchases </p><p><strong>from</strong> </p><p>customer\_licence </p><p><strong>where</strong> </p><p>is\_successfully\_converted is true and (settlement\_date between '2023-07-01' and '2023-08-01') </p><p><strong>group by</strong> </p><p>licence\_type;</p>                                                                                                                                                                                                                                                                          |
| Total Number of Active Subscribers, by licence type | <p><strong>select</strong> </p><p>licence\_type , count(distinct customer\_exid) as total\_active\_subscribers </p><p><strong>from</strong> </p><p>customer\_licence </p><p><strong>where</strong> </p><p>licence\_type in ('SUBSCRIPTION','RENTAL','FIXED\_DATE') and ( licence\_start\_date is not null and licence\_start\_date < '2023-08-01' and (licence\_end\_date is null or (licence\_start\_date != licence\_end\_date AND licence\_end\_date > '2023-07-01')) )<br><strong>group by</strong> </p><p>licence\_type;</p>                                                                                                 |
| Total Number of Active Licences, by licence type    | <p><strong>select</strong> </p><p>licence\_type , count(order\_id) as total\_active\_licences </p><p><strong>from</strong> </p><p>customer\_licence </p><p><strong>where</strong> </p><p>licence\_type in ('SUBSCRIPTION','RENTAL','FIXED\_DATE') and ( licence\_start\_date is not null and licence\_start\_date < '2023-08-01' and (licence\_end\_date is null or (licence\_start\_date != licence\_end\_date AND licence\_end\_date > '2023-07-01')) )<br><strong>group by</strong> </p><p>licence\_type;</p>                                                                                                                  |
| Licences Due to churn (Pending Churn)               | <p><strong>select</strong> </p><p>count(order\_id) </p><p><strong>from</strong> </p><p>customer\_licence </p><p><strong>where</strong> </p><p>licence\_status in ('ACTIVE\_CHURN', 'TRIAL\_ACTIVE\_CHURN');</p>                                                                                                                                                                                                                                                                                                                                                                                                                   |
| Total promo code redemptions                        | <p>For New Purchases:<br><br><strong>select</strong> count(order\_id) as total\_promo\_redemptions <br><strong>from</strong> <br>customer\_licence <br><strong>where</strong> <br>is\_voucher\_used is true or is\_gift\_card\_used is true or first\_promotional\_campaign\_id\_used is not null;<br><br>For All Payment Transactions:<br><br><strong>select</strong> <br>count(order\_id) as total\_promo\_redemptions <br><strong>from</strong> <br>revenue <br><strong>where</strong> <br>txn\_type = 'DEBIT' and ( is\_voucher\_used is true or is\_gift\_card\_used is true or promotional\_campaign\_id is not null );</p> |
| Total number of IAP New Purchases                   | <p><strong>select</strong> <br>licence\_type , count(order\_id) as total\_new\_succ\_purchases <br><strong>from</strong> <br>customer\_licence <br><strong>where</strong> <br>is\_successfully\_converted is true and (settlement\_date between '2023-07-01' and '2023-08-01') and payment\_provider in ('ROKU\_IAP', 'MICROSOFT\_IAP', 'APPLE\_IAP', 'GOOGLE\_IAP', 'AMAZON\_IAP') <br><strong>group by</strong> <br>licence\_type;</p>                                                                                                                                                                                          |
| Average Churn in the last 30 days                   | <p><strong>select</strong> <br>avg(churned\_licences) as avg\_churn <br><strong>from</strong> ( <br><strong>select</strong> date(licence\_end\_date) as day , count(order\_id) as churned\_licences <br><strong>from</strong> <br>customer\_licence <br><strong>where</strong> <br>licence\_type in ('FIXED\_DATE','SUBSCRIPTION','RENTAL') and (licence\_end\_date between date(utc\_now() - INTERVAL '30 days') and date(utc\_now() + INTERVAL '1 days')) <br><strong>group by</strong> <br>date(licence\_end\_date) );</p>                                                                                                     |
| Total Involuntarily Churned Licences                | <p><strong>select</strong> <br>count(order\_id) as churned\_licences <br><strong>from</strong> <br>customer\_licence <br><strong>where</strong> <br>licence\_type in ('FIXED\_DATE','SUBSCRIPTION','RENTAL') and (licence\_end\_date between '2023-07-01' and '2023-08-01') and is\_voluntary\_churn is false;</p>                                                                                                                                                                                                                                                                                                                |
| Total number of free trial licences                 | <p><strong>select</strong> <br>count(order\_id) as total\_free\_trial\_licences <br><strong>from</strong> <br>customer\_licence <br><strong>where</strong> <br>started\_as\_free\_trial is true;</p>                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Total licences in free trial                        | <p><strong>select</strong> <br>count(order\_id) AS total\_licences\_in\_free\_trial <br><strong>from</strong> </p><p>customer\_licence <br><strong>where</strong> <br>licence\_status in ('TRIAL\_ACTIVE','TRIAL\_ACTIVE\_CHURN');</p>                                                                                                                                                                                                                                                                                                                                                                                            |
| Total converted free trial licences                 | <p><strong>select</strong> <br>count(order\_id) as total\_free\_trial\_converted\_licences <br><strong>from</strong> <br>customer\_licence <br><strong>where</strong> <br>started\_as\_free\_trial is true and is\_free\_trial\_converted is true;</p>                                                                                                                                                                                                                                                                                                                                                                            |

## Revenue Metrics

<table data-full-width="true"><thead><tr><th>Metric</th><th>SQL</th></tr></thead><tbody><tr><td>Total Revenue</td><td><p><strong>select</strong> </p><p>round(sum(amount_usd)/100.0, 2)::numeric(20,2) as total_revenue </p><p><strong>from</strong> </p><p>revenue </p><p><strong>where</strong> </p><p>txn_type = 'DEBIT' and settlement_date between '2023-07-01' and '2023-08-01';</p></td></tr><tr><td>LTV</td><td></td></tr><tr><td>Average Revenue per User</td><td><p><strong>select</strong> round(sum(amount_usd)/100.0/count(distinct customer_exid), 2)::numeric(20,2) as avg_revenue_per_user </p><p><strong>from</strong> </p><p>revenue </p><p><strong>where</strong> </p><p>txn_type = 'DEBIT' and settlement_date between '2023-07-01' and '2023-08-01';</p></td></tr><tr><td>Average Revenue per New Purchase</td><td><p><strong>select</strong> round(sum(amount_usd)/100.0/count(order_id), 2)::numeric(20,2) as avg_revenue_per_new_purchase from ( select order_id , settlement_date , amount_usd , row_number() over (partition by order_id order by txn_seq) AS rn </p><p><strong>from</strong> </p><p>revenue </p><p><strong>where</strong> </p><p>txn_type = 'DEBIT' and is_pending is false ) as txns_of_first_purchases where rn = 1 and (settlement_date between '2023-07-01' and '2023-08-01')</p></td></tr><tr><td>Total Revenue Refunded</td><td><p><strong>select</strong> </p><p>round(sum(amount_usd)/100.0, 2)::numeric(20,2) as total_revenue </p><p><strong>from</strong> </p><p>revenue </p><p><strong>where</strong> </p><p>txn_type = 'CREDIT' and settlement_date between '2023-07-01' and '2023-08-01';</p></td></tr><tr><td>Total number of Refunds</td><td><p><strong>select</strong> </p><p>count(order_id) as total_number_of_refunds </p><p><strong>from</strong> </p><p>revenue </p><p><strong>where</strong> </p><p>txn_type = 'CREDIT' and settlement_date between '2023-07-01' and '2023-08-01';</p></td></tr></tbody></table>
