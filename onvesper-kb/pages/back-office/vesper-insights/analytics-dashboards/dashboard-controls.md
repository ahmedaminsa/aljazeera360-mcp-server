# Dashboard Controls

<figure><img src="/files/KJd8kQ55xuGwHOCmhWVx" alt=""><figcaption></figcaption></figure>

Navigable items live along the top side menu of Vesper Insights

<figure><img src="/files/00YpmKwLbZgoh1rGAw9W" alt=""><figcaption></figcaption></figure>

Use the top left area to navigate between sheets showing different analysis sets

[Filters](/platform-knowledge-base/back-office/vesper-insights/analytics-dashboards/dashboard-controls/filters.md) allow you to dive deeper into the data

<figure><img src="/files/PjvSL7NnoIjjCABkryjq" alt=""><figcaption></figcaption></figure>

[Segments](/platform-knowledge-base/back-office/vesper-insights/analytics-dashboards/dashboard-controls/segments.md) allow you to set a group of filters and monitor users that fall into that set of criteria at the desired time window

<figure><img src="/files/spdvCCInwKtSrryGhUIY" alt=""><figcaption></figcaption></figure>

Once activated, Segments allow you to use [Targeting](/platform-knowledge-base/back-office/vesper-insights/analytics-dashboards/dashboard-controls/targeting.md) to send the specific set of qualifying customers data to 3rd party marketing partners.&#x20;

<figure><img src="/files/2xuTq5WmDVfeA8ywOhIz" alt=""><figcaption></figcaption></figure>

[Cohort](/platform-knowledge-base/back-office/vesper-insights/analytics-dashboards/dashboard-controls/cohort.md) allows you to identify a group of users that fall into an set of initial criteria then modify filters around them to monitor that specific group over time/status changes

<figure><img src="/files/6L5rjl0qEhuJJmZrA80T" alt=""><figcaption></figcaption></figure>

[Export](/platform-knowledge-base/back-office/vesper-insights/analytics-dashboards/dashboard-controls/export.md) will download a one sheet image of the dashboards currently in view

<figure><img src="/files/Z65902ZJMeCmZSo1v74l" alt=""><figcaption></figcaption></figure>

[Download](/platform-knowledge-base/back-office/vesper-insights/analytics-dashboards/dashboard-controls/download.md) will queue a CSV export of the underlying data as relates to the current filter state

<figure><img src="/files/FwZZazYo0Ssptv4cT14u" alt=""><figcaption></figcaption></figure>

Reset the dashboard to default filters

<figure><img src="/files/doo7tXGNfUzeoYubrkvP" alt=""><figcaption></figcaption></figure>

Apply any filter changes to the current dashboard
