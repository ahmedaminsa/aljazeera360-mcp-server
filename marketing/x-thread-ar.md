**1/ 🧵**

عملت أول MCP server للمحتوى العربي 🇸🇦

دلوقتي تقدر تربط Claude أو ChatGPT أو Cursor بمكتبة **الجزيرة 360** كاملة — وثائقيات، بودكاست، برامج، حلقات — وتسأل بالعربي وترجّع روابط مشاهدة مباشرة.

مفتوح المصدر ومجاني 👇

---

**2/**

يعني إيه بالظبط؟

تقول للمساعد: «ايه أحدث حلقات الدحيح؟» أو «دوّرلي على وثائقيات عن غزة»

المساعد يتصل بالسيرفر، يجيب البيانات الحقيقية من الجزيرة 360، ويرجّعلك العناوين والمدة ورابط المشاهدة على aljazeera360.com مباشرة.

---

**3/**

أسهل طريقة — من غير أي تثبيت:

فيه نسخة مستضافة جاهزة. تضيف السطرين دول في إعدادات المساعد وخلاص:

```json
{
  "mcpServers": {
    "aljazeera360": {
      "type": "http",
      "url": "https://aljazeera360-mcp-server-production.up.railway.app/mcp"
    }
  }
}
```

**4/**
