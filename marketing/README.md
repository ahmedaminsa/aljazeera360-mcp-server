# Marketing kit

Launch content for the Al Jazeera 360 MCP Server. See [`../LAUNCH.md`](../LAUNCH.md)
for the full checklist and posting sequence.

| File | Use |
| :--- | :--- |
| [`x-thread-ar.md`](x-thread-ar.md) | Arabic X/Twitter thread — lead channel |
| [`x-thread-en.md`](x-thread-en.md) | English X/Twitter thread |
| [`show-hn.md`](show-hn.md) | Show HN title, URL, first comment, talking points |
| [`devto-post.md`](devto-post.md) | Long-form blog post (dev.to / Hashnode) |
| [`reddit.md`](reddit.md) | r/ClaudeAI, r/LocalLLaMA, and other community posts |

**One rule before you post anything:** connect the hosted instance in Claude and
run a few real queries first, so the analytics dashboard has data when curious
people check it.
